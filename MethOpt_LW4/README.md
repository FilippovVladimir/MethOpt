# Задание №4. Решение задачи динамического программирования

**Ф.И.О.:** Филиппов Владимир Леонидович    
**Поток:** МетОпт 1.2  

## Описание

Проект реализует решение задачи оптимизации инвестиционного портфеля методом динамического программирования с использованием критерия Байеса для принятия решений в условиях неопределенности.

## Структура проекта

- `portfolio_optimization.py` - основная программа с реализацией алгоритма
- `report.md` - подробный отчет с математической постановкой задачи

## Запуск программы

```bash
python portfolio_optimization.py
```

Либо в среде программирования.

## Результаты

Программа выводит:
- Максимальный ожидаемый доход
- Оптимальную последовательность действий на каждом этапе
- Состояния портфеля после каждого действия
- Возможные исходы после каждой ситуации

## Метод решения

Используется метод динамического программирования с обратным прохождением (backward induction) и рекуррентным соотношением Беллмана. Для оптимизации вычислений применяется мемоизация.

## Критерий принятия решений

Используется критерий Байеса (максимизация математического ожидания дохода).
