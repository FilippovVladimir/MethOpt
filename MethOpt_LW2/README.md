# Задание №2. Метод ломаных

**Ф.И.О.:** Филиппов Владимир Леонидович    
**Поток:** МетОпт 1.2  

## Цель работы
Реализовать программу, которая находит **глобальный минимум** липшицевой функции `f(x)` на отрезке `[a,b]`
с заданной точностью `eps`, используя **метод Пиявского (метод ломаных)**.

## Вход и выход
**Вход:**
- строка функции,
- `a`, `b` — границы отрезка,
- `eps` — точность.

**Выход:**
- график функции `f(x)`,
- вспомогательная ломаная (нижняя оценка по методу Пиявского),
- точка найденного минимума,
- `x*`, `f(x*)`,
- число итераций и время,
- PDF с демонстрацией.

## Файлы в репозитории
- `lab2.ipynb` — ноутбук с кодом и пояснениями.
- `lab2_demo.pdf` — PDF-демонстрация (график + результат).

## Как запустить
1) Откройте `lab2.ipynb`.
2) Выполните ячейки сверху вниз.
3) В блоке “Запуск” запускается `test_6` (Растригин) и сохраняется PDF `lab2_demo.pdf`.

## Демонстрация
Используется `test_6` — **функция Растригина** (имеет много локальных минимумов).
